Production rules

Format locks

Send from
Ann's name (not brand)
Format
Plain text feel, no banner image, no CTA button
Link
Inline link in email 3 only
Image
Optional blurred/silhouette product shot in 1+2. Crisp shot in 3 only.
Platform
Klaviyo single sends
Audience
Existing customers + email subscribers (segment exclusive)
Cadence
Day 0 → Day 3-4 → Day 6-7 → Launch Day 8-9
Reply tracking
Email 1+2 invite replies. Count guesses, callout patterns in email 2.

Locks needed before send

  • Exact launch date (so Day-X math holds)
  • Refill pack size confirmed (assumes 500ml, change if wrong)
  • Travel-size ml (not quoted yet, add if you want it specific)
  • Confirm $39 bundle price + free-travel-bottle language
  • Confirm "limited first run" framing is true (don't promise scarcity if stock plentiful)
  • Set launch [DAY DATE] + [TIME] placeholders in email 3 body
  • Klaviyo segment built: existing customers + subscribers, excluded from EOFY sale list

Why this sequence works

  • → Travel bottle = easy guess (people half-know). Refill = hard guess (genuinely new). Tension drives reply rate.
  • → Reply invites in 1+2 = list warm-up before launch send. Klaviyo engagement score climbs.
  • → "Better than the sale" anchor preserves EOFY scarcity message. Different audience, different offer architecture.
  • → Limited first run = real scarcity (not manufactured), justifies exclusivity.
  • → Founder voice + plain text = matches Mother's Day email pattern that worked.

Burned / don't do

  • Subject lines starting "Big news" / "Drumroll" / "Guess what?" (too marketing-y)
  • Em dashes (Ann scrub rule, use commas/periods/"and")
  • Big hero banner image (kills plain-text feel)
  • CTA button on email 1+2 (kills the tease)
  • Reveal product in email 1 image (defeats the guessing mechanic)
  • Hard sell language in email 3 ("DON'T MISS OUT", "Buy now")
  • Identity-led Type C / Certified Chaos copy in subject lines (save for body)
  • Sending to full list — exclusive offer is the whole hook, don't dilute
