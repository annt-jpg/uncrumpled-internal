How the pack runs

Two phases · one offer

Phase 1 · Lead gen (C01) runs 7-14 days before sale go-live. Promise = early access + first dibs on 25% off in exchange for email signup. Anchored on the "we only do two sales a year, next one's November" scarcity hook. 4 visual styles so the algorithm finds the cheapest signup. Phase 2 · Sale live (C02) runs 7 days at 25% off sitewide, including the new launches (refill pack + travel bottle). 10 ad styles rotating 5 angles — plain discount, "your sign", scarcity, new launches, urgency — so we can read which copy converts best on cold and warm.

Designer direction: the on-creative copy table per card is the only copy that goes on the image. Primary text, headline, description and CTA button live in Meta Ads Manager (see the pink Meta Spec table). Photo asset map below uses 9 hero shots Ann supplied. Where new-product imagery is referenced (refill / travel), the current photo is a placeholder — sub in new SKU photography once shot. Lockup chrome: cream #F5EFE2 bg · sage deep #5C6E50 band/CTA · ink #2C2A24 type · lowercase serif uncrumpled wordmark top-left every mock. Sticker chip = circular sage tag rotated -8deg, mid-right. No top-right corner pill chips (RO designer rule).

Production · designer brief

Locks · flags · burned list

LOCKSformat spec
Aspect4:5 (1080×1350) primary. Also export 1:1 (1080×1080) + 9:16 (1080×1920).
WordmarkFraunces lowercase, top-left, every mock. White on photo, ink on cream band.
PaletteCream #F5EFE2 · Sage deep #5C6E50 · Ink #2C2A24
Sticker chipSage circle, rotated -8°, mid-right. NOT top-right (RO designer rule).
TypographyFraunces serif for headlines + chip · Inter sans for sub + CTA
Sale anchor"Annual sale" / "two sales a year" / "next one's November"
Discount line"25% off sitewide" (locked, not 30%)
NEW pill colourPop red #B8504A only for new-launch mocks (SL-06, SL-08, SL-09)
FLAGSresolve before launch
Discount25% locked here. Existing private 2026-05-07 pack was 30%. Reconcile.
DatesSale start + end TBC. Drives placeholders + SL-10 countdown.
RRP for was/nowSL-03 uses "save 25%" framing only — exact $ pricing not yet locked.
Travel bottle photoSL-06 + SL-09 use 250ml as placeholder. Need real 100ml product shot.
Refill pack photoSL-08 uses 250ml as placeholder. Need real refill pouch photo.
Lead gen LPBuild presale signup form before LG ads go live.
SuppressionAudience that submitted via LG ads → exclude from SL cold retargeting.
NamingApply Uncrumpled Meta naming convention to all 14 ads before publish.
BURNEDdo not use
Phrasing"Don't miss out" · "Hurry" · "Today only" · "Once in a lifetime"
Compound hyphens"best-deal" / "anti-perfection" hyphenated. Use as 2 words or skip hyphen.
Em dashesAnn scrub rule. Use commas, periods, or "and".
Corner chipsNo top-right pill chips. RO designer treats them as production assets.
Split screensNo Type B vs Type C diptychs. Killed 2026-05-01.
Identity in saleNo Type C / Certified Chaos copy in sale-lane mocks. Save for organic.
Fake countdownsNo animated timers in static creative.
Negative compNo knocking competitor categories (irons, steamers). Reframe positively.
